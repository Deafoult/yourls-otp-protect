# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https.keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https.semver.org/spec/v2.0.0.html).

## [0.3] - 2025-11-10

### Added
- [SECURITY] Add settings to change the QR-Code generator (e.g. link/placeholder).
- Add settings to change  OTP length and algorithm.

## [0.2] - 2025-11-09

### Added
- Initial release of the OTP Protect plugin.
- Secure YOURLS instances so that only people with a valid OTP secret can create new short URLs.
- Settings page to manage OTP secrets.
- QR code generation for easy setup in authenticator apps.
